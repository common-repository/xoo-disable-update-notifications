Disables Theme and Plugin update reminders selectively.

== Description ==

An intuitive and easy-to-use plugin that lets you selectively disable plugin and theme update reminders in the Wordpress Admin interface.

= All features =

* Choose yourself which plugins and themes to disable update reminders for.
* Enable back at any time, no data is ever lost.

= Additional features available in Xoo Disable Updates Pro =

* Priority support
* Disable at multisite admin level for all sites on network

Buy <a href="http://xoocode.com/xoo-disable-updates/">Xoo Disable Updates Pro here</a>.

== Installation ==

1. Go to Plugins -> Add New
2. Search for Xoo Disable Updates.
3. Download and install.
4. Activate the plugin.
5. Xoo Disable Updates is now available as a submeny under the 'Xoo' main menu in the wordpress admin interface left menu (below the 'Settings' menu).

You can also install this plugin manually:

1. Download the plugin's ZIP archive and unzip it.
2. Copy or move the unzipped 'Xoo Disable Updates' folder to the /wp-content/plugins/ directory.
3. Activate the plugin through the Plugins menu in WordPress

== Uninstalling ==

Simply deactivate or remove the plugin from Plugins -> installed plugins page. There is no risk in deactivating/deleting the plugin.
